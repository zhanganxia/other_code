JS的三种书写方式：行内式、嵌入式、外链式（script）
注意：外链式  一个script标签如果做了一个外链一个js文件的操作后，
    它里面就不要再写js代码类，如果真的要写，单独在写一个。

定义一个变量，如果没有赋值，是undefined
变量和函数的命名规范：
    1.严格区分大小写；
    2.首字符：字目 下划线（_）$
    3.除了首字符之外的其他字符：数字 字￥母 下划线$


函数的预解析，当执行函数如果发现的函数还没有定义，它会字段向下去找，如果后面有函数的
定义，此函数也能正常执行；

if 语句的条件需要用括号括起来
    if(){

    }else if(){

    }

'==' 它会先把左右两边的变量转换成相同，之后再做比较
例如：iNum = 15 , sStr = '15'
    iNum == sStr  -->成立

'===' 不会转换类型，如果类型不一样肯定不相等

“&&” 一假则为假 只要有一个不成立就不成立（与）

“||” 一真则为真 只要有一个条件成立就成立（或）

“ ！” 取反


如果一个函数只执行一次就可以用匿名函数
如果ig函数需要在多个地方去调用，就不要用匿名函数
